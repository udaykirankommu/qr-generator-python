# QR Code Generator

A simple Python project that generates QR codes from text or URLs and saves them as PNG images.

## Concepts Used

* Variables
* Input/Output
* Strings
* String Concatenation
* Functions
* Modules
* Third-Party Libraries
* Object Methods
* File Handling
* f-Strings

## Requirements

```bash
pip install qrcode[pil]
```

## How to Run

```bash
python qr.py
```

## Example

Input:

```text
Enter text or URL: https://www.google.com
Enter file name(without .png): google
```

Output:

```text
QR Code generated successfully!
saved as: google.png
```

A file named `google.png` will be created in the project folder.
